# web-g6

## Docker Compose
プロジェクトルートで以下のコマンド
- start
  ```
  docker-compose up --build
  ```
- stop
  ```
  別のターミナルを起動して
  docker-compose down
  ```
  - or
    ```
    Ctrl + C 連打
    docker-compose down
    ```
